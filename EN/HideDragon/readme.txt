 				《隐身专家》自述文件
				~~~~~~~~~~~~~~~~~~~~
				=-=-=-=-=-=-=-=-=-=-=
                                 欢迎使用《隐身专家》
                                =-=-=-=-=-=-=-=-=-=-=
《隐身专家》是一个快速隐藏桌面和任务栏的窗口的程序。
如果你正在做一些不想让你的老师、老板...看到的操作，
那么这个程序正好适应你。它可以快速隐藏桌面上所有的窗
口，等合适的时候你再恢复。

包含文件：
~~~~~~~~~

	1.QuickHide.exe                应用程序文件
	2.QuickLib.dll                 库文件
        3.QuickHide.exe.manifest       Windows XP皮肤文件
        4.Misc.hdo                     设置保存

运行环境：
~~~~~~~~~
    Windows 9x/Me/2000/XP

安装:
~~~~~
    《隐身专家》不需要安装，直接拷贝到你的硬盘上就可以了，
不想用时直接删掉就可以。

使用方法：
~~~~~~~~~
    本程序的使用方法非常简单，运行QuickHide.exe，如果有不需要被隐藏的
窗口，在“窗口列表”列表框里选中，点击“保留”按钮，最小化程序。程序
最小化后在任务栏里看不到，在托盘图标区域里有一个三条黄线的小图标，鼠标
左键单击一下就可以显示程序主窗口，右键单击会有快捷菜显示。然后就继续的
做你自己的事吧，如果看到你的老师走过来了，就一齐按下鼠标的左右键(双手
在键盘就按Ctrl + H)，这样就可以隐藏不希望他看到的程序了，等他走过去了，
再按下鼠标左右键或Ctrl + H，刚刚隐藏的窗口又显示出来了。

注意事项：
~~~~~~~~~
    * 点"选项"就可以对《隐身专家》进行设置，如键盘快捷键，刷新窗口列表
      的时间间隔。不推荐使用手动刷新，如果在保留窗口时忘了手动刷新程序
      有时会出错。

    * 当程序退出时被隐藏的窗口如何处理：有两个选项，一个是全部显示，一个
      是结束所有进程。默认是全部显示。

    * 为了防止别人看到你在运行《隐身专家》，程序有一个功能就是隐藏托盘图
      标，要隐藏托盘图标可以在程序主窗口点"隐藏图标"，或在图标上点右键，在
      弹出的快捷菜单上选"隐藏图标"。这时要显示《隐身专家》的主窗口要按快
      捷键Ctrl + W，或再运行QuickHide.exe，这时主窗口就会显示出来了。
    
    * 在设置快捷时"隐藏桌面所有窗口"的快捷键和呼出《隐身专家》主窗口的快捷
      键不可以相同。
